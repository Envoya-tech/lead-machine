  • Installing system dependencies
    Homebrew + Xcode Command Line Tools — this is the slowest part, can take 5–10 minutes on a fresh Mac. Your machine is not frozen.
  • Setting up Python environment
    Installing Python 3.11 and creating an isolated virtual environment for Lead Machine.
  • Installing database
    Downloading and installing PostgreSQL — the database that stores your leads and campaigns.
  • Creating database
    Setting up the Lead Machine database and initialising its structure.
  • Installing application
    Downloading Lead Machine's Python packages and applying database migrations.
  • Starting services
    Launching the backend server and web proxy. These will run automatically every time you log in.
  • Almost ready…
    Waiting for Lead Machine to respond — this usually takes a few seconds.

⚠ Taking longer than expected

The backend service didn't start in time. Open Terminal and check the error log:

cat ~/Library/Logs/LeadMachine/backend-error.log

Or check the full install log:

tail -50 /tmp/leadmachine-install.log

Common fix: reinstall the package, then wait a few minutes before checking this page again.
